# Quick Search App

A simple Android app for multi-platform quick search.

## Features

- Search across 5 platforms: XiaoHongShu, Bilibili, Zhihu, Bing, Google
- XiaoHongShu, Bilibili, Zhihu open directly in their apps
- Bing and Google open in Edge browser
- Clean dark UI

## Build APK

This project uses GitHub Actions to automatically build APK.

### Steps:

1. Create a new repository on GitHub
2. Upload all files to the repository
3. Go to Actions tab
4. Wait for the build to complete
5. Download the APK from the Artifacts section

## Requirements

- Flutter SDK 3.16.0+
- Android SDK
